# Speedometer

Mobile speedometer built with React, Vite and Tailwind CSS.

## Setup

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

### Lint

```bash
npm run lint
```

### Test

```bash
npm test
```
