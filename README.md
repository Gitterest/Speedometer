# Speedometer

Mobile speedometer built with React, Vite and Tailwind CSS.

## Setup

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

Toggle premium mode in `src/utils/featureFlags.js`.
